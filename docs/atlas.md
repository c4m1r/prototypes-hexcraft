# Маппинг текстур для атласа

Атлас: `src/game/texutres.png` (320x320 пикселей, блоки 32x32)

## Структура атласа

### Ряд 1 (Y=0): Основные блоки
- **Col 0 (X=0)**: grass-top - верхняя текстура травы
- **Col 1 (X=32)**: grass - боковая текстура травы (трава сверху, земля снизу)
- **Col 2 (X=64)**: dirt - земля (верх и бок одинаковые)
- **Col 3 (X=96)**: stone - камень (верх и бок одинаковые)
- **Col 4 (X=128)**: sand - песок (верх и бок одинаковые)
- **Col 5 (X=160)**: snow - снег (верх и бок одинаковые)
- **Col 6 (X=192)**: ice - лёд (верх и бок одинаковые, полупрозрачный)
- **Col 7 (X=224)**: bronze - бронза (для будущего использования)
- **Col 8 (X=256)**: silver - серебро (для будущего использования)
- **Col 9 (X=288)**: gold - золото (для будущего использования)

### Ряд 2 (Y=32): Анимированные блоки (по 3 кадра)
- **Col 0-2 (X=0-96)**: lava1, lava2, lava3 - лава (анимация)
- **Col 3-5 (X=96-192)**: water1, water2, water3 - вода (анимация, полупрозрачная)

### Ряд 3 (Y=64): Ресурсы
- **Col 0 (X=0)**: wood - дерево (верх и бок одинаковые)
- **Col 1 (X=32)**: leaves - листва (верх и бок одинаковые, полупрозрачная)

### Ряд 4 (Y=96): Природа
- **Col 0 (X=0)**: red mushroom - красный гриб
- **Col 1 (X=32)**: mushroom - коричневый гриб

## Маппинг блоков игры

### Текущие блоки (BLOCK_TYPES):

1. **grass** (Трава)
   - Top: Row 0, Col 0 (grass-top)
   - Side: Row 0, Col 1 (grass)

2. **dirt** (Земля)
   - Top: Row 0, Col 2 (dirt)
   - Side: Row 0, Col 2 (dirt)

3. **stone** (Камень)
   - Top: Row 0, Col 3 (stone)
   - Side: Row 0, Col 3 (stone)

4. **sand** (Песок)
   - Top: Row 0, Col 4 (sand)
   - Side: Row 0, Col 4 (sand)

5. **water** (Вода)
   - Top: Row 1, Col 3-5 (water1-3, анимация)
   - Side: Row 1, Col 3-5 (water1-3, анимация)
   - Прозрачность: 0.7

6. **wood** (Дерево)
   - Top: Row 2, Col 0 (wood)
   - Side: Row 2, Col 0 (wood)

7. **leaves** (Листва)
   - Top: Row 2, Col 1 (leaves)
   - Side: Row 2, Col 1 (leaves)
   - Прозрачность: 0.75

8. **snow** (Снег)
   - Top: Row 0, Col 5 (snow)
   - Side: Row 0, Col 5 (snow)

9. **ice** (Лёд)
   - Top: Row 0, Col 6 (ice)
   - Side: Row 0, Col 6 (ice)
   - Прозрачность: 0.9

10. **lava** (Лава)
    - Top: Row 1, Col 0-2 (lava1-3, анимация)
    - Side: Row 1, Col 0-2 (lava1-3, анимация)

## Примечания

- Анимированные блоки (lava, water) используют 3 кадра для анимации
- Полупрозрачные блоки (ice, water, leaves) требуют правильной настройки прозрачности
- Для шестигранных блоков верхняя и боковая текстуры могут отличаться (например, grass)
- Текстуры загружаются через TextureManager и применяются в режиме Modern
